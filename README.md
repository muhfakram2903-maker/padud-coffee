# ☕ Padud Coffee

**Padud Coffee** adalah sebuah aplikasi mobile untuk pemesanan menu digital berbasis pemindaian QR Code. 

Aplikasi ini dirancang untuk memudahkan pelanggan dalam melihat daftar menu, melakukan pemesanan, serta menyelesaikan pembayaran secara mandiri (self-service) langsung dari perangkat smartphone mereka tanpa perlu antre di kasir. 

Dengan memindai QR Code yang ada di setiap meja, pelanggan dapat menikmati pengalaman pemesanan yang lebih cepat, praktis, dan modern, sekaligus membantu operasional kafe menjadi lebih efisien.
